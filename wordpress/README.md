# How to Use the Sanmati Journal Redesigned Archive Page Template on WordPress & Elementor

This folder contains the complete standalone redesigned archive page for the Sanmati Journal:
* **[archive-template.html](file:///Users/aradhyjain/Library/CloudStorage/GoogleDrive-jainaradhy01@gmail.com/My%20Drive/sanmati-journal/wordpress/archive-template.html)**: Self-contained HTML file containing full styling, icons, JSON database of all 80 papers, live search, and filters.

---

## 🚀 Deployment Instructions

### 1. Elementor (Recommended)
1. Log in to your WordPress dashboard.
2. Go to **Pages > Add New** (or edit your existing `/archive` page) and click **Edit with Elementor**.
3. In the left panel search bar, search for the **HTML** widget.
4. Drag and drop the **HTML Code** widget into your desired column/section.
5. Copy the entire contents of **[archive-template.html](file:///Users/aradhyjain/Library/CloudStorage/GoogleDrive-jainaradhy01@gmail.com/My%20Drive/sanmati-journal/wordpress/archive-template.html)**.
6. Paste the code into the **HTML Code** text area in Elementor.
7. Click **Update** or **Publish** at the bottom.

---

### 2. WordPress Block Editor (Gutenberg)
1. Go to **Pages > Add New** (or edit your existing `/archive` page).
2. Click the **+** button to add a new block.
3. Search for and select the **Custom HTML** block.
4. Open **[archive-template.html](file:///Users/aradhyjain/Library/CloudStorage/GoogleDrive-jainaradhy01@gmail.com/My%20Drive/sanmati-journal/wordpress/archive-template.html)**, copy all contents, and paste them into the Custom HTML block text area.
5. Click **Publish** or **Update**.

---

### 3. Classic Editor (TinyMCE)
1. Go to **Pages > Add New** (or edit your existing `/archive` page).
2. Switch from the **Visual** tab to the **Text** (HTML) tab at the top right of the text editor.
3. Paste the entire content of **[archive-template.html](file:///Users/aradhyjain/Library/CloudStorage/GoogleDrive-jainaradhy01@gmail.com/My%20Drive/sanmati-journal/wordpress/archive-template.html)**.
4. Click **Publish** or **Update**.

## 🛠️ Features Included:
* **Premium Design**: Built using the official colors (`#0F4C81`, `#2563EB`, `#F59E0B`, `#F8FAFC`) and Poppins/Inter typography.
* **Statistics Counter**: Interactive boxes display total volumes, issues, and papers.
* **Spotlight Banner**: Prominent top highlight banner for the inaugural volume PDF compendium.
* **Instant Live Search**: JavaScript search by title, author name, or volume updating instantly without reloading the page.
* **Volume/Author Filtering**: Dropdowns allow readers to filter by volume or select a specific author dynamically.
* **Responsive Layouts**: Renders as clean research tables on desktop and table structures transform into beautiful compact cards on mobile phones.
* **Custom Dropdowns**: Hover-actions for View PDF and Download PDF.
